# Mouse Movement to Transform Node (for DaVinci Resolve)

A web-tool that creates a Transform node for DaVinci Resolve from the movement of your mouse.

Use this for character animation, memes, puppet show animations and more!

[>>> Open Mouse Movement to Transform Node](https://neezr.github.io/Mouse-Movement-to-Transform-Node/)

## Usage
- [On the website](https://neezr.github.io/Mouse-Movement-to-Transform-Node/), select your screen size and draw into the box
    - *(The screen size has no effect on the node, but will just resize the canvas.)*
- After that, your mouse movement will be converted to a Transform node.
    - Copy from the text box at the bottom of the site or use the Copy button
    - Paste into DaVinci Resolve Fusion!
